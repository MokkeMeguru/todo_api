## 🎯 このプロジェクトの特徴

### Rust言語の活用
- **型安全性**: コンパイル時にエラーを検出
- **メモリ安全性**: 所有権システムによる保証
- **パフォーマンス**: ゼロコスト抽象化
- **並行性**: 安全な非同期処理

### クリーンアーキテクチャ
- **ドメイン層**: ビジネスロジックの核心
- **ユースケース層**: アプリケーションのビジネスルール
- **インターフェース層**: 外部システムとの接続
- **インフラストラクチャ層**: 技術的な実装詳細

### OpenAPI統合
- **API仕様駆動開発**: OpenAPI仕様からコードを自動生成
- **型安全性**: 生成されたコードによる型保証
- **ドキュメント自動生成**: Swagger UIによるAPIドキュメント

## 🛠️ 技術スタック

| 技術 | 用途 |
|------|------|
| **Rust** | メイン言語 |
| **Axum** | Webフレームワーク |
| **Tokio** | 非同期ランタイム |
| **Serde** | シリアライゼーション |
| **OpenAPI Generator** | コード生成 |
| **Utoipa** | APIドキュメント生成 |
